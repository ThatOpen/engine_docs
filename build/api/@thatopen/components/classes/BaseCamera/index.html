abstract BaseCamera

Abstract class representing a camera in a 3D world. All cameras should use this class as a base.

controls?​

optional abstract controls: CameraControls

Optional CameraControls instance for controlling the camera. This property is only available if the camera is controllable.

currentWorld: null | World = null

The current world this item is associated with. It can be null if no world is currently active.

onWorldChanged​

readonly onWorldChanged: Event<object>

Event that is triggered when a world is added or removed from the worlds map. The event payload contains the world instance and the action ("added" or "removed").
