# LeaderAnnotation

The committed data for a single leader annotation.

## Properties

### arrowTip

> **arrowTip**: `Vector3`

Tip of the arrow — the annotated point.

***

### elbow

> **elbow**: `Vector3`

Elbow — the bend between the leader line and the extension.

***

### extensionEnd

> **extensionEnd**: `Vector3`

End of the horizontal extension line — anchor for the text.

***

### style

> **style**: `string`

Name of the [LeaderAnnotationStyle](LeaderAnnotationStyle.md) to use when rendering.

***

### text

> **text**: `string`

The annotation text.

***

### uuid

> **uuid**: `string`

Unique identifier.
