# LodMode

Enum representing the mode of the LOD / culling system.

## Enumeration Members

| Enumeration Member | Value | Description |
| :------ | :------ | :------ |
| `ALL_GEOMETRY` | `2` | Hides invisible items, displays the rest as full geometry |
| `ALL_VISIBLE` | `1` | Displays all items as full geometry |
| `DEFAULT` | `0` | Hides invisible items, displays far away items as LOD geometry, displays close items as full geometry |
