# RawTransformData

> **RawTransformData**: `object`

Data defining a transform (local or global) of a mesh.

## Type declaration

### position

> **position**: `number`[]

The position of the transform.

### xDirection

> **xDirection**: `number`[]

The x direction of the transform coordinates.

### yDirection

> **yDirection**: `number`[]

The y direction of the transform coordinates.
